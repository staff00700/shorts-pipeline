Shorts Uploader

Automated short-form video publishing tool

About

Shorts Uploader is a self-hosted server application that creates and publishes educational short-form videos (30–60 seconds) to social media platforms. It downloads public domain educational content, processes clips with subtitles, and uploads them on a daily schedule.

Features

  • Automated video downloading and processing with FFmpeg
  • AI-powered moment selection from video transcripts
  • Embedded subtitle generation via Whisper
  • Scheduled daily publishing (up to 5 shorts/day)
  • Web admin panel for monitoring and comment management
  • Cross-platform support: YouTube, TikTok
